Installation & Setup

1

Download the Installer

Click the download button above to get the `ThunderCut_Installer.exe`. This bundled package contains both the core engine and the interface.

2

Run & Permissions

Double-click the installer. If Windows SmartScreen appears, click "More Info" and "Run Anyway". The software requires administrator rights to access Serial ports.

3

Connect Your Hardware

Plug in your cutting plotter via USB. Open ThunderCut, go to 'Device Settings', and click 'Auto-Detect'. You're ready to start cutting!
